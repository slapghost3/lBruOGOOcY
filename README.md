# 仓库管理系统 python394

## 项目概述

仓库管理系统是基于 Django 框架开发的一套业务管理系统。该系统致力于简化仓库管理流程，提供高效便捷的用户操作界面。

## 技术栈

- Python 3.7
- Django
- MySQL 5.7
- Vue.js

## 功能模块

### 系统角色

- **管理员**
 - 系统首页
 - 个人中心
 - 用户管理
 - 公告信息管理
 - 产品信息管理
 - 产品类型管理
 - 产品入库管理
 - 产品出库管理
 - 产品借出管理
 - 事件记录管理

- **用户**
 - 系统首页
 - 个人中心
 - 公告信息管理
 - 产品信息管理
 - 产品入库管理
 - 产品出库管理
 - 产品借出管理
 - 事件记录管理

## 运行环境

- Python 3.7 环境配置
- MySQL 5.7 数据库
- Vue.js 前端环境

## 部署步骤

1. 配置 Python 3.7 环境及依赖库
2. 安装并配置 MySQL 5.7 数据库
3. 部署 Django 后端服务
4. 部署 Vue.js 前端服务

## 目录结构

```
project/
│
├── app/ # Django 应用目录
│ ├── migrations/ # 数据库迁移文件

├── static/ # 静态资源文件

├── manage.py # 管理脚本
└── ...
```

- 基于成熟的 Django 框架，提供稳定的系统性能。
- 完善的权限控制，区分管理员与用户角色，确保数据安全。
